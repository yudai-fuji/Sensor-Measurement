# Sensor Measurement

Androidデバイス（スマートフォンやスマートグラス等）に搭載されている各種センサーのデータをリアルタイムで取得・表示し、CSVファイルとしてローカル保存できる計測アプリケーションです。

## 概要
本アプリは、デバイスの動作や環境データを計測するためのツールです。
取得したセンサーデータはタイムスタンプとともに画面上でリアルタイム確認できるほか、計測終了後に一括で外部ストレージ（Documentsディレクトリ）にCSV形式で書き出すことが可能です。
また、画面遷移時（`onPause`）にセンサーリスナーを自動解除しない仕様となっており、バックグラウンドでの計測継続にも対応しています。

## 主な機能
- **リアルタイムデータ表示**: 加速度、線形加速度、ジャイロスコープ、ゲーム回転ベクトル、気圧の各センサー値を画面上で確認できます。
- **データ記録機能**: `Start` ボタン押下から `Stop` ボタン押下までの全センサーイベントデータを内部バッファに記録します。
- **CSVファイル出力**: 記録したデータをワンタップでCSVファイルとして書き出せます。
- **ストップウォッチ表示**: 計測の経過時間（分:秒:ミリ秒）をリアルタイムで画面表示します。

## 対応センサー一覧
本アプリで計測・記録対象としているセンサーおよび出力フォーマットは以下の通りです。
（※端末側にハードウェアとして搭載されていないセンサーのデータは取得されません）

| センサー名 | Android Sensor Type | CSV識別子 | 取得データ・単位 |
| :--- | :--- | :--- | :--- |
| **加速度センサー** | `TYPE_ACCELEROMETER` | `acc` | X, Y, Z軸の加速度 (m/s²) |
| **線形加速度センサー** | `TYPE_LINEAR_ACCELERATION` | `Lacc` | X, Y, Z軸の加速度 ※重力加速度を除く (m/s²) |
| **ジャイロスコープ** | `TYPE_GYROSCOPE` | `Gyro` | X, Y, Z軸の角速度 (rad/s) |
| **ゲーム回転ベクトル** | `TYPE_GAME_ROTATION_VECTOR` | `GameRo` | クォータニオン値 (X, Y, Z, W) |
| **気圧センサー** | `TYPE_PRESSURE` | `pre` | 環境気圧 (hPa) |

## 動作要件・ビルド環境
- **最小SDKバージョン (minSdk)**: API Level 31 (Android 12以上)
  - ※コード内の設定通り、スマートグラス等にデプロイする場合は用途に合わせて `build.gradle.kts` の `minSdk` を `29` 等に変更してください。
- **ターゲットSDKバージョン (targetSdk)**: API Level 35
- **コンパイルSDKバージョン (compileSdk)**: API Level 35
- **Java互換性**: JavaVersion 11

## 使用方法

### 1. 計測の開始
- アプリを起動すると待機画面が表示されます。
- **`Start`** ボタンをタップすると計測が開始され、ストップウォッチのカウントアップと画面上のセンサー値の更新が始まります。

### 2. 計測の停止
- **`Stop`** ボタンをタップするとセンサーの記録が停止し、計測時間が確定します。

### 3. データの書き出し (保存)
- 計測停止後、データが存在する場合に **`Save`** ボタンが有効になります。
- タップすると端末の外部ストレージ内の `Documents` フォルダ配下にCSVファイルが自動生成されます。
  - **出力先パス例**: `内部ストレージ/Documents/SensorData_YYYYMMDD_HHmmss.csv`

### 4. リセット
- **`Reset`** ボタンをタップすると、画面表示、ストップウォッチ、および内部に保持されている計測データがクリアされ、初期状態に戻ります。
